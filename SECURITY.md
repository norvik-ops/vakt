# Security Policy

## Unterstützte Versionen

Sicherheitsupdates werden für die jeweils aktuell veröffentlichte Version bereitgestellt.

| Version | Unterstützt |
|---------|-------------|
| Latest  | ✅          |
| Älter   | ❌          |

## Sicherheitslücke melden

**Bitte erstelle kein öffentliches GitHub Issue für Sicherheitslücken.**

Melde Sicherheitsprobleme vertraulich per E-Mail:

**security@norvikops.de**

Bitte nenne:
- Art der Lücke (z.B. XSS, Auth-Bypass, IDOR)
- Betroffene Komponente und Version
- Schritte zur Reproduktion
- Mögliche Auswirkungen

Wir melden uns in der Regel innerhalb von 48 Stunden.
